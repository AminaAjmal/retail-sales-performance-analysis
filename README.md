# Retail Sales Performance Dashboard (Excel)

## Project Overview
This project focuses on analyzing retail sales data to identify profitability drivers, regional performance disparities, and customer segment purchasing patterns. Using Microsoft Excel, I transformed raw transactional data into an interactive, stakeholder-ready executive dashboard.

## Key Insights & Business Recommendations
* **Regional Disparities:** The **West region** generated the highest overall sales ($725K), while the **South region** underperformed significantly ($392K). 
* **Profitability Drainage:** While **Technology** was the most profitable category globally (~$145K profit), **Tables** (under the Furniture category) in the South region showed negative profit margins. *Recommendation: Audit regional supplier agreements, adjust shipping configurations, or recalibrate pricing tiers for underperforming Furniture items in the South.*
* **Customer Segments:** **Consumer** clients represent the largest driver of business volume, accounting for approximately 51% of total sales.

## Technical Skills Demonstrated
* **Data Cleaning & Engineering:** Handled missing values, formatted data constraints, and prepared data pipelines in a dedicated `Clean Data` sheet.
* **Aggregations:** Built advanced logic using **Pivot Tables** to segment variables across regions, products, and categories.
* **Data Visualization:** Developed optimized user charts (Bar, Pie, Donut, and Horizontal Ranking charts) leveraging a cohesive professional color palette.
* **Interactivity:** Implemented multi-table connected **Slicers** to allow dynamic exploratory analysis for non-technical stakeholders.
